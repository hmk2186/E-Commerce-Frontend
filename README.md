# 🛍️ E-Commerce Frontend

A modern, responsive E-Commerce Frontend built using **HTML5, CSS3, Bootstrap and Vanilla JavaScript**. The project provides a complete shopping experience with product browsing, product details, wishlist, shopping cart, and responsive UI across different devices.

## 🚀 Live Demo

🔗 https://e-commerce-frontend-alpha-mauve.vercel.app/

---

## ✨ Features

- 🏠 Responsive Home Page
- 🛒 Shopping Cart
- ❤️ Wishlist
- 📦 Product Listing
- 🔍 Product Details Page
- 📱 Fully Responsive Design
- 📞 Contact Page
- ℹ️ About Page
- 🎨 Clean and Modern UI
- ⚡ Fast and Lightweight
- 💻 Built using Vanilla JavaScript (No Framework)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Local Storage

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/hmk2186/E-Commerce-Frontend-with-Admin-Panel.git
```

Go to the project folder

```bash
cd E-Commerce-Frontend-with-Admin-Panel
```

Open `index.html` in your browser.
